<h1>Using the EPBot</h1>
<h2>The repository contains examples of how to use the functions implemented in the EPBot dll.</h2>

## Features

There are three EPBot dlls available:
- EPBot86.dll
- EPBot64.dll
- EPBotARM64.dll

All of them contain EPBot as namespace.
The included .tlb files allow you to use COM.
